# AI Job Market Trends Analysis 
This project explores 50,000 job postings to understand what drives salary and hiring in AI.
##  Project Highlights
Statistical Validation: I used Bootstrap Resampling ($10,000$ iterations) to calculate a 95% confidence interval for salary differences based on specific skills.

Machine Learning: Implemented K-Means Clustering to group skills into 6 distinct categories based on Education, Experience, and Remote-work flexibility.

Data Visualization: Includes 3D cluster plots and industry-skill heatmaps.
## 🛠 How to Run
Clone the repo.

Install dependencies: pip install -r requirements.txt.

Run Project.ipynb.
